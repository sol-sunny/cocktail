:point_right:cocktail Api를 불러와 원하는 칵테일을 검색하고 원하는 칵테일을 선택하여 아이템을 확인할 수 있는 앱

## :memo:기능
+ 유저가 input에 값을 입력한다
+ input값과 일치하는 칵테일을 찾을 수 있다.
+ 아이템을 클릭하면 상세페이지를 확인할 수 있다.
